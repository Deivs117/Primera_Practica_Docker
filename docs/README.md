# docs/

Esta carpeta contiene la documentación de referencia de la práctica.

## Documento guía original

El documento de la práctica original (`Practica_Contenedores_Docker.docx` o
equivalente) debe ser colocado aquí por el instructor o el alumno.

> **Nota**: El binario del documento no se incluye en el repositorio para mantener
> el tamaño del repo pequeño. Solicita el documento a tu instructor.

## Contenido de referencia

El `README.md` principal del repositorio contiene todos los comandos y
explicaciones necesarios para completar:

- Partes 1–6 de la práctica
- Ejercicios 1–4

Los "Desafíos" (puntos 5–7 del documento original) **no están implementados**
en este repositorio.
